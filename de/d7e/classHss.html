DiagStatus_t Hss::diagRead ( float  amps,
float  iisFault,
float  iisOl,
uint16_t  kilis,
Channel_t  ch = NO_CHANNEL 
)

Diagnosis of the Sensor.

This function is using the IS signal to determine the state of the switch. It returns an diagnosis state of the switch.

Parameters
[in]ampsSensed current value
[in]iisfaultSensed current at fault condition
[in]iisOlOpen load detection threshold
[in]kilisCurrent sense ratio
[in]chChannel no. (*Optional)
Returns
DiagStatus_t
Return values
-2Not enabled
0Switch is working fine
1Overload detected
5Open load detected
Note
This function should be called only after you get the Is value. Also note, in case you are using shield with no channel differentiation, then ignore the 'ch' parameter and this will default to NO_CHANNEL.
